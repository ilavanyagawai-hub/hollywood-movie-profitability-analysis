🎬 Hollywood Movie Profitability & Performance Analysis

## 📌 About the Project

This project is an end-to-end data analysis of Hollywood movies using Microsoft Excel, PostgreSQL, SQL, and Tableau. The objective was to analyze movie profitability, worldwide gross collections, audience ratings, and studio performance to understand the key factors that contribute to a movie's success.

The project follows a complete analytics workflow—from data cleaning and SQL analysis to building an interactive Tableau dashboard that helps explore movie performance across different genres, studios, and years.

---

## 🎯 Project Objectives

- Analyze overall movie profitability.
- Compare profitability across different genres.
- Identify the highest-grossing movie genres.
- Find the top 10 most profitable movies.
- Analyze studio-wise average profitability.
- Study the relationship between Audience Score and Rotten Tomatoes ratings.
- Track the number of movies released each year.

## 💡 Key Insights

- Drama movies recorded the highest average profitability.
- Comedy movies generated the highest worldwide gross.
- Profitability varied significantly across movie studios.
- Audience scores showed a positive relationship with Rotten Tomatoes ratings.
- A few blockbuster movies contributed a large share of overall profitability.
